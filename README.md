# AndroidGamesResearch

**Open Research in AI for Games, Computational Game Analytics, and Software Engineering**

A collection of open research, datasets, software, and reproducible resources supporting research in **AI for Games**, **computational game analytics**, **data-centric AI**, and **empirical software engineering**.

The repositories include published research, open datasets, analytical tools, and ongoing research projects that promote reproducible research and collaboration between academia and industry.

---

## Research Areas

* 🎮 AI for Games
* 📊 Computational Game Analytics
* 🤖 Data-Centric AI
* 👥 Player Behaviour Analytics
* 📱 Mobile Game Analytics
* 🔍 Explainable AI
* 🧠 Machine Learning
* 💻 Empirical Software Engineering

---

## Published Research

### 📚 TTAG+R: Dataset of Trending Android Games with User Reviews

A publicly available dataset of Google Play's top trending Android games and user reviews for research in game analytics, machine learning, and software engineering.

### 📚 Dataset of Software Quality Attributes

Research datasets supporting empirical studies on software quality assessment and software analytics.

### 📚 Mapping Android Features to Quality Attributes

Research exploring relationships between Android application features and software quality attributes using data-driven approaches.

### 📚 Visual Ethics for UI/UX in Android Game Apps

Research on visual ethics and user interface design in Android game applications.

---

## Ongoing Research

* **GameMetaDroid** *(under development)* – A large-scale dataset supporting AI-driven mobile game analytics and computational game analytics research.
* **Interaction Analysis for Game Analytics** *(in progress)* – Investigating interaction-aware learning and interpretable models for understanding game success.

## Mission

To develop and share high-quality datasets, research software, and reproducible resources that advance AI for Games, game analytics, and software engineering research.
